# Student Management System (Java)

## 📌 Project Description
This is a simple console-based Java application to manage student records. It allows users to perform basic operations like adding, viewing, updating, and deleting student data.

## 🚀 Features
- Add Student
- View Students
- Update Student
- Delete Student
- Menu-driven interface

## 🛠️ Technologies Used
- Java
- OOP (Object-Oriented Programming)
- ArrayList (Collections Framework)

## ▶️ How to Run
1. Open terminal in project folder
2. Compile:
   javac StudentManagementSystem.java
3. Run:
   java StudentManagementSystem

1. Add Student

4. Delete Student
